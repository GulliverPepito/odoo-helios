Payroll Analytic Account from Contract

Setting to allow Payroll Analytic Account from Contract or Salary Rule.
by Intellego-BI.com

This addon will enable Payslip debit postings to inherit the Analytic Account set on the Employee Contract, according to the configuration of each Salary Rule.

Setting for Analytic Account for each Salary Rule:


When field* is not set, Analytic Account may be set directly on Salary Rule and will ignore the setting on Employee Contract.

field* = "Cuenta Analítica por Contrato"


When field is set, Analytic Account is not visible on Salary Rule and debit postings will take Analytic Account from Employee Contract.


Example of Accounting Journal for payroll posting. Note that the Analytic Account is only transfered to debit postings which represent the payroll expenses that are now sliced by Analytic Account.
