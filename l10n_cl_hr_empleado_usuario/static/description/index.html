Chile - RRHH Empleados desde Usuarios Odoo

[l10n_cl_hr_empleado_usuario]

Automaticamente crea Empleado al crear un Usuario Odoo.

Funciones

☑ Vincula Usuario Odoo con Empleado
☑ Crea Contacto con Dirección Personal para Empleado (pago nómina)
☑ Sincroniza cambios entre Usuario y Empleado
☑ No requiere configuración ni acciones adicionales
☑ Al crear un usuario Odoo se debe indicar si es usuario Interno o Empleado
☑ Los campos obligatorios para Empleados se ingresan al crear el Usuario

Autores

Intellego-BI.com

Resumen

Permite descentralizan el otorgamiento de préstamos a empleados y automatizar los flujos de trabajo asociados.

Dependencias

  • Odoo - Empleados [hr]
  • Odoo - Nómina [hr_payroll]
